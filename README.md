# Praktikum RPL 01

## Fishing Spot Sharing System
Sebuah aplikasi untuk membantu para pemancing (_angler_) mencari dan berbagi spot memancing sesuai kebutuhan mereka.

## Fitur Utama
1. Pencarian spot memancing berdasarkan lokasi dan preferensi.
2. Berbagi informasi spot oleh pengguna.
3. Informasi dasar spot (jenis ikan, waktu terbaik, dan akses lokasi).

## Tech Stack (Rencana)
- Frontend: Web (akan ditentukan pada iterasi implementasi)
- Backend: API service (akan ditentukan pada iterasi implementasi)
- Database: Relational/NoSQL (akan ditentukan pada iterasi implementasi)

## Struktur Repository
- `docs/` : Dokumentasi tim dan workflow.
- `src/` : Sumber kode aplikasi.
- `tests/` : Dokumen/skenario pengujian.

## Workflow Pengembangan
Workflow lengkap ada di: `docs/workflow.md`

Ringkasan:
1. Buat branch fitur dari `dev` dengan format `feature/nama-fitur`.
2. Lakukan perubahan kecil dan commit dengan pesan yang jelas.
3. Push branch lalu buat Pull Request ke `dev`.
4. Lakukan review, revisi, dan merge setelah disetujui.

## Cara Menjalankan
Proyek masih pada tahap inisialisasi dokumentasi.  
Panduan setup dan eksekusi aplikasi akan ditambahkan saat implementasi kode dimulai.
